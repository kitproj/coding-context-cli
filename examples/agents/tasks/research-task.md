# Research Task

You are tasked with researching a technical topic and providing a comprehensive analysis.

## Research Topic

- **Topic**: ${research_topic}
- **Context**: ${research_context}

## Your Task

1. **Investigate and Analyze**
   - Research the topic thoroughly
   - Review existing implementations
   - Compare different approaches
   - Identify pros and cons

2. **Document Findings**
   - Create comprehensive documentation
   - Include code examples where relevant
   - Cite sources and references
   - Provide clear recommendations

3. **Present Options**
   - List viable approaches
   - Compare trade-offs
   - Recommend best option with rationale
   - Note risks and considerations

## Guidelines for Research

- **DO NOT** implement production code
- **DO NOT** write tests
- **DO** provide example code snippets for clarity
- **DO** document everything thoroughly
- **DO** cite sources and references
- **DO** consider multiple perspectives

## Output

Provide:
- Research findings document
- Comparison of approaches
- Recommended solution with justification
- Implementation considerations
- Potential risks and mitigation strategies
